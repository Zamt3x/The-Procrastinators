About the Project

As with many project, ours uses a collection of different components, working together to form what is seen on screen. This section is dedicated to shed some light on the inner workings of Marvin, and what was used to achieve the end result.

Marvin's 3D model

The 3D model of Marvin was made by the user safa9raja3 and was gotten from Free3D.com.

Further texture modifications were done by us, and rigging (creating a model skeleton for animations) + animations are from Adobe's Mixamo.

Marvin's brain

His brain uses modified data files from Richard Wallace (drwallace), which can be found on GitHub, utilized by the aiml package from PyPi.

Marvin's brain also uses medical data about symptoms connected to illnesses, gotten from people.dbmi.columbia.edu, then used by the ChatterBot package from PyPi.

Together these two components are used by a pre-compiled Python script to generate Marvin's responses and decide the final output.

Behind the scenes and interface

The project itself is based on the NodeJS JavaScript runtime + Electron, making use of the power of HTML, CSS and JavaScript to achieve markup, styling and logic.
