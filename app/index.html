About the Project

As with many project, ours uses a fair bit of different components from all around the internet. This section is dedicated to shed some light on the inner workings of Marvin, and what was used to achieve the end result.

Marvin's voice

The TTS (Text-To-Speech) technology uses the standard JavaScript SpeechSynthesis API. If it sounds a little weird, it's because Chromium doesn't have access to the same amount of different voices that Google Chrome has, and is probably using your Operating System's default voice.

Marvin's 3D model

The 3D model of Marvin was made by the user safa9raja3 and was gotten from Free3D.com.

Further texture modifications were done by us, and rigging (creating a model skeleton for animations) + animations are from Adobe's Mixamo.

Marvin's brain

His brain uses modified data files from Richard Wallace (drwallace) and can be found on GitHub.

Marvin's brain also uses medical data about symptoms connected to illnesses, gotten from people.dbmi.columbia.edu.

Together these two components are used by a pre-compiled Python script to generate Marvin's responses.

Behind the scenes and interface

The project itself is based on NodeJS and Electron, making use of the power of JavaScript, HTML and CSS to power both logic, markup and looks.
